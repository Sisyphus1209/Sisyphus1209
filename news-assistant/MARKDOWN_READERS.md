# Markdown 阅读器推荐

## 🥇 首选推荐：Obsidian（免费）

**为什么推荐：**
- ✅ 完全免费（个人使用）
- ✅ 支持中文完美
- ✅ 颜值高，阅读体验极佳
- ✅ 可以建知识库，把新闻都存进去
- ✅ 支持图谱视图，看新闻关联

**下载地址：**
https://obsidian.md/download

**使用方法：**
1. 下载安装
2. 创建新仓库（Vault），选择 `news_assistant/news_reports` 文件夹
3. 所有新闻报告自动显示在左侧文件列表
4. 点击即可阅读，支持搜索、标签、链接

---

## 🥈 极简美观：MarkText（免费开源）

**为什么推荐：**
- ✅ 完全免费开源
- ✅ 界面极简，类似 Typora
- ✅ 实时预览，所见即所得
- ✅ 无广告，轻量级

**下载地址：**
https://github.com/marktext/marktext/releases
（下载 .exe 安装包）

---

## 🥉 开发者首选：VS Code + 插件

如果你已经有 VS Code：

**安装插件：**
1. 打开 VS Code
2. 扩展商店搜索 "Markdown All in One"
3. 安装后即可完美预览 .md 文件

**优点：**
- 免费
- 可以编辑也可以阅读
- 支持大纲视图

---

## 💎 颜值巅峰：Typora（付费但值得）

**特点：**
- 最美观的 Markdown 编辑器
- 所见即所得（输入时直接看到效果）
- 一次性买断 $14.99

**下载地址：**
https://typora.io/

**免费替代品：** MarkText（上面推荐的）

---

## 🌐 不想安装软件？用浏览器！

把 .md 文件拖到浏览器，或安装扩展：

**Chrome/Edge 扩展：**
- "Markdown Viewer"
- "Markdown Preview Plus"

---

## 我的建议

| 场景 | 推荐 |
|------|------|
| 想建立新闻知识库 | **Obsidian** |
| 只想简单阅读 | **MarkText** |
| 已经有 VS Code | 装个插件就行 |
| 愿意为颜值付费 | **Typora** |

## 快速开始（Obsidian）

1. 下载安装 Obsidian
2. 打开后点击 "打开文件夹作为仓库"
3. 选择：`C:\Users\Administrator\news_assistant\news_reports`
4. 左侧看到所有新闻文件
5. 点击任意文件即可阅读

**效果：**
- 中文显示完美
- 表格、表情符号都正常
- 可以搜索关键词
- 可以给新闻打标签
